Descrição do Projeto

Sistema Provedor 

O sistema provedor tem como obejtivo facilitar a organização de um provedor de internet e controlar a entrada e saida de seus clientes,
no qual,o sistema cadastra os dados do cliente, tais como: nome, telefone, endereço e a quantidade de megas que o cliente solicitou,
daí o software calcula quanto o cliete deverá pagar pela quantidade de megas contratado,o mesmo cadastra novos clientes, salvando-os
em seu banco de dados,podendo assim, pesquisar os dados dos clientes cadastrados e alterar caso necessário, o sistema também poderá 
deletar 
Para que o sistema execute,o facilitador da empresa deverá informar(cadastrar) os dados do cliente (nome,telefone,
endereço e quantidade de megas desejado), fazendo este cadastro, o sistema já calcula altomaticamente quanto o cliente 
deverá pagar pelo serviço e se caso o cliente não queira mais os serviços, através do sistema o facilitador poderá deletar o 
cliente cadastrado.
