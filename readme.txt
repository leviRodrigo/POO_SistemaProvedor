Descrição do Projeto

SPEED NET

O programa SPEED NET tem como obejtivo facilitar a organização de um provedor de internet e controlar a entrada e saida de seus clientes,
no qual,o sistema cadastra os dados do cliente, tais como: nome, telefone,cpf, endereço e a quantidade de megas que o cliente solicitou,
daí o software calcula quanto o cliete deverá pagar pela quantidade de megas contratado,o mesmo cadastra novos clientes, salvando-os
em seu banco de dados,podendo assim, pesquisar os dados dos clientes cadastrados e alterá-los caso necessário, o sistema também poderá 
deletar o cliente cadastrado, caso ele não queira mais os serviços da empresa, assim, o dono da empresa terá um controle de entrada e saída de seus clientes. 
Para que o sistema execute,o facilitador da empresa deverá informar(cadastrar) os dados do cliente (nome,telefone,cpf
endereço e quantidade de megas desejado), fazendo este cadastro, o sistema já calcula altomaticamente quanto o cliente 
deverá pagar pelo serviço e caso o cliente não queira mais os serviços, através do sistema o facilitador poderá deletar o 
cliente cadastrado, tendo assim total controle de entra e saída dos clientes da empresa.
